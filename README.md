# 技术栈

前端的布局与样式全部使用的bootstrap，动作全部使用jquery。

后端采用MVC架构，控制层使用反射优化了结构。
使用了apache的dbutil通过结果集处理器直接转化为list，并使用c3p0连接池优化数据库访问。

前后通信前端使用ajax发送请求, 端使用jackson以结果类的形式将请求的数据封装为json返回。
